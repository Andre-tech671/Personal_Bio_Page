# Andre Philip Nyanjahia - Personal Bio Page

A personal portfolio website showcasing the professional profile, skills, experience, and projects of Andre Philip Nyanjahia, a Principal Full-Stack Engineer & Solutions Architect based in Tete, Mozambique.

## 🚀 Overview

This is a responsive personal bio page built with HTML, CSS, and Font Awesome icons. It serves as a digital resume and portfolio showcasing Andre's extensive experience in full-stack development, data science, Web3 technologies, and social work leadership.

## 🛠️ Technologies Used

- **HTML5** - Semantic markup for structure
- **CSS3** - Styling and responsive design
- **Font Awesome 6.5.1** - Icons for visual enhancement
- **Google Fonts** - Typography (Arial, sans-serif)

## 💻 How to View

Simply open the `docs/index.html` file in any modern web browser:

```
bash

open docs/index.html

# Option 2: Using a local server (for better compatibility)
npx serve .
```

## 📝 Customization

To customize this template:

1. Replace `docs/assets/images/Andre_Business_Photo.png` with your own photo
2. Update the content in `docs/index.html` with your personal information
3. Modify the CSS styles in the `<style>` section as needed
